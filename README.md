# SafeVault Security Enhancement Project Using Microsoft Copilot

## Repository Overview

This repository contains the deliverables for the SafeVault Security Enhancement Project. The project demonstrates secure coding practices, authentication and authorization mechanisms, vulnerability remediation, and security testing with assistance from Microsoft Copilot.

## Security Features

### Input Validation

* Username validation
* Email validation
* Malicious input detection

### SQL Injection Prevention

* Parameterized SQL queries
* User input handling through query parameters

### Authentication

* Password hashing using SHA256
* Credential verification
* Secure login validation

### Authorization

* Role-Based Access Control (RBAC)
* User role permissions
* Administrative access restrictions

### Vulnerability Remediation

* SQL Injection mitigation
* Cross-Site Scripting (XSS) mitigation
* Broken Access Control mitigation

### Security Testing

* Input validation tests
* SQL injection prevention tests
* Authentication tests
* Authorization tests

## Microsoft Copilot Usage

Microsoft Copilot was used throughout the project to:

* Generate secure code examples.
* Assist with authentication and authorization logic.
* Identify and fix security vulnerabilities.
* Generate security-focused test cases.
* Improve overall code quality and maintainability.
